# 验证计划

## 必做验证

1. 覆盖验证：每个传感器每日有效水体覆盖率。
2. 传感器一致性：同日或近同日清水像元的 FAI/NDVI 偏差。
3. 阈值敏感性：FAI 阈值在 0.03-0.10 之间变化时面积是否稳定。
4. 留一日期验证：融合估计与真实高分观测对比。
5. QA 面积分层：真实观测、融合、粗分、插值分开统计。

## 评价指标

| 指标 | 用途 |
|---|---|
| FAI RMSE/MAE | 连续指数误差 |
| Bloom precision/recall/F1 | 二值水华识别 |
| IoU | 水华空间重叠 |
| Area relative error | 面积估计是否可用 |
| Peak date difference | 事件峰值日期是否可信 |
| QA coverage ratio | 结论由多少真实观测支撑 |

## 通过标准

融合层只有满足以下条件才进入正式逐日 30 m 结论：

- 至少有足够的高分观测日期用于留一验证，建议不少于 10 天。
- 关键水华事件日前后有高分或粗分独立证据支撑。
- 面积误差和 IoU 达到报告预设阈值。
- 每日统计表明确列出 QA=10、20、30、40、90 的比例。

否则应退回两层报告：高分观测日空间格局 + MODIS/Sentinel-3 日尺度趋势。
